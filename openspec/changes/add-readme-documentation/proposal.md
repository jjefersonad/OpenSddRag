## Why

O projeto OpenSddRag não possui documentação de usuário final. Novos usuários e colaboradores não conseguem entender rapidamente como instalar, configurar e usar a ferramenta sem ler o código-fonte diretamente. Um README.md detalhado resolve essa lacuna e aumenta a adoção.

## What Changes

- Adição de `README.md` na raiz do repositório com documentação completa de uso
- Cobertura de instalação, configuração, comandos CLI, transporte MCP, e fluxo SDD

## Capabilities

### New Capabilities
- `readme-documentation`: Documento README.md completo com guia passo a passo de instalação, configuração do banco de dados, uso do mcp-server (CLI e servidor), uso do client Node.js, e visão geral do fluxo SDD (propose → apply → archive)
